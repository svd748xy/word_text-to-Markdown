使用指引

目前支持以下文件格式的转换：

.docx — Microsoft Word 文档（2007及以上版本），使用 mammoth.js 引擎解析
.txt — 纯文本文件，UTF-8 和 GBK 编码均可自动识别
.html — HTML 网页文件，使用 turndown.js 引擎转换

暂不支持旧版 .doc 格式（二进制格式），请先将 .doc 另存为 .docx 后再上传。
转换引擎会尽力保留原始文档的结构和格式：

标题层级（H1-H6）、段落与换行
无序列表和有序列表
表格（仅 .docx 和 .html 支持）
图片引用（保留图片的 alt 文本和路径信息）
超链接
粗体、斜体、删除线等文本格式
代码块和行内代码

注意：复杂的嵌套表格、自定义样式和某些 Word 专有格式可能无法完美保留。
单个文件大小限制为 30MB，超过此限制的文件将被拒绝。

普通文档（几MB以内）的转换通常在 1-3秒 内完成。较大的文档（接近30MB）可能需要5-10秒。

本工具支持同时上传多个文件并逐个转换，您可以一次性处理多个文档。
完全安全。本工具的所有处理均在您的浏览器本地完成，不会将文件上传到任何服务器。文件内容仅在您的设备内存中处理，转换完成后即可关闭页面清除所有数据。
转换完成后，点击底部的 打包下载全部 (.zip) 按钮，即可将所有已转换的 Markdown 文件打包为一个 ZIP 文件下载。您也可以点击每个文件卡片上的下载图标单独下载某个文件。
